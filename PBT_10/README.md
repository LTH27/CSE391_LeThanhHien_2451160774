# PBT_10 - Async JavaScript & API Integration

Folder chứa các bài tập PBT_10 theo đề bài.

Các app:
- `weather_app/` — Weather App dùng Open-Meteo API
- `user_directory/` — CRUD user với JSONPlaceholder API
- `gallery/` — Infinite scroll gallery với JSONPlaceholder Photos
- `dashboard/` — Multi-API dashboard với Random User, Dog CEO, REST Countries

Mỗi thư mục có `index.html`, `style.css`, `app.js`, `README.md`.
