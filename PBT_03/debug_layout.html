🔍 Debug Layout: Box Model Problem

Bài tập C1 — Tìm hiểu tại sao layout bị vỡ khi sử dụng float và cách khắc phục bằng box-sizing hoặc tính toán lại kích thước.

❌ Section 1: Layout Bị Vỡ Broken

Sử dụng content-box mặc định — padding và border được cộng thêm vào width.

⚠️ Layout bị vỡ! Content bị đẩy xuống dòng mới vì tổng width vượt quá container.

📄 CSS gây lỗi

.container { width: 960px; margin: 0 auto; }
.sidebar  { width: 300px; padding: 20px; border: 1px solid #ccc; float: left; }
.content  { width: 660px; padding: 30px; border: 1px solid #ccc; float: left; }

📐 Tính toán kích thước thực tế

Sidebar thực tế
300 + 20×2 + 1×2 = 342px
width + padding + border
Content thực tế
660 + 30×2 + 1×2 = 722px
width + padding + border
Tổng cộng
342 + 722 = 1064px
❌ 1064px > 960px — Vượt quá container!

✅ Section 2: Fix 1 — box-sizing: border-box Fixed

Padding và border được tính bên trong width → tổng width không thay đổi.

✅ Layout hiển thị đúng! Sidebar và Content nằm cạnh nhau hoàn hảo.

📄 CSS đã sửa

.sidebar, .content {
  box-sizing: border-box;  /* ← Thêm dòng này */
}
.container { width: 960px; margin: 0 auto; }
.sidebar  { width: 300px; padding: 20px; border: 1px solid #ccc; float: left; }
.content  { width: 660px; padding: 30px; border: 1px solid #ccc; float: left; }

📐 Tính toán kích thước thực tế

Sidebar thực tế
300px (bao gồm tất cả)
padding + border nằm trong 300px
Content thực tế
660px (bao gồm tất cả)
padding + border nằm trong 660px
Tổng cộng
300 + 660 = 960px
✅ 960px = 960px — Vừa khít container!

✅ Section 3: Fix 2 — Tính toán lại width Fixed

Giữ content-box mặc định nhưng trừ đi padding và border từ width ban đầu.

✅ Layout hiển thị đúng! Không cần border-box, chỉ cần tính toán chính xác.

📄 CSS đã sửa

.container { width: 960px; margin: 0 auto; }
.sidebar  { width: 258px; padding: 20px; border: 1px solid #ccc; float: left; }
              /* 300 - 20×2 - 1×2 = 258px */
.content  { width: 598px; padding: 30px; border: 1px solid #ccc; float: left; }
              /* 660 - 30×2 - 1×2 = 598px */

📐 Tính toán kích thước thực tế

Sidebar thực tế
258 + 20×2 + 1×2 = 300px
Đúng mục tiêu 300px ban đầu
Content thực tế
598 + 30×2 + 1×2 = 660px
Đúng mục tiêu 660px ban đầu
Tổng cộng
300 + 660 = 960px
✅ 960px = 960px — Vừa khít container!
